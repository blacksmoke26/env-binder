# Changelog

## version 0.0.1 (2025-11-XX)

* Added type-safe environment variable management
* Implemented parsing for JSON, Cron expressions, Patterns, and Enums
* Added support for Numbers, Integers, and URLs with intelligent parsing
* Introduced nested objects and arrays handling
* Added complex types support (arrays, objects, dates, URLs, time durations)
* Implemented time units parsing (minutes, hours, days, weeks)
* Added validation and parsing for various data types
* Introduced custom transformation functions
* Added required fields support with error throwing
* Implemented Boolean falsy values and string arrays support
* Added path/url alias resolution with `$PREFIX` support
* Implemented singleton pattern for consistent configuration access
* Added variable substitution in environment values
* Included runtime type validation and error handling
* Added beautiful TypeScript intellisense support
